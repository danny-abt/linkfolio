# LinkFolio

Portfolio personnel de Danny Emmanuel Aboutou — étudiant en génie logiciel à l'Université Laval.

## Stack

- **Next.js 15** — React framework avec SSG
- **TypeScript** — Typage statique
- **Tailwind CSS 4** — Utility-first CSS
- **Framer Motion** — Animations déclaratives

## Développement

```bash
npm install
npm run dev
```

## Déploiement

Déployé sur [Vercel](https://vercel.com) — push sur `main` déclenche un déploiement automatique.
